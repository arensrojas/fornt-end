# Guia de trabajo en clases

## Paso a paso para trabajar con Git
1. Crear archivo prueba.txt
2. Agregar texto a prueba.txt
3. Inicializar repositorio 'git init'
4. Agregar al área de preparación (staging area) 'git add'
5. Agregar al repositorio 'git commit -m "mensaje descriptivo"'
6. Hacer cambios en el archivo
7. 'git add .' y 'git commit -m "mensaje"'

## Definiciones
- git -version: Muestra la versión instalada de Git
- git init: Crea un repositorio Git en la carpeta actual
- git add: Prepara archivos para ser incluidos en el siguiente commit
- git commit: Guarda los cambios agregados con un mensaje
- git status: Muestra el estado del repositorio
  - Archivos modificados
  - Archivos listos para el commit
  - Archivos no rastreados
- git push: Sube los commits al repositorio remoto
- git pull: Trae todos los cambios desde Github y los mezcla con los tuyos
- git clone: Clona un repositorio remoto a tu computador
- git log: Muestra el historial de los commits
